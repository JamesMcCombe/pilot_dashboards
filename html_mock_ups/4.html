Data Window

What We Can Observe Today

Even without broker mandates, a small but meaningful behavioural window exists through voluntary integrations that NZ traders already use.

04
Opt-in integration sources
  • NinjaTrader bridge for prop and futures-style trading workflows.
  • MT4 / MT5 connectivity via existing APIs and plugins.
  • Interactive Brokers API for NZ-based investors and traders.
Behavioural signals available
  • Trade timing and frequency across sessions and instruments.
  • Leverage usage and position sizing behaviour.
  • Stop-loss usage and other basic risk controls.
  • Patterns of account destruction and recovery attempts.

This is not a full-market dataset — it is an indicator dataset that still reveals powerful patterns in NZ trader behaviour.
